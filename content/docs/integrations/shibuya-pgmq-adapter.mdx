title: shibuya ⇄ pgmq adapter
description: Consume a pgmq message queue through shibuya's supervised pipelines — leasing, retries, and an optional dead-letter queue.
---

This adapter lets [shibuya](/docs/shibuya) consume a [pgmq](/docs/pgmq) message
queue, exactly parallel to the [shibuya ⇄ kiroku adapter](/docs/integrations/shibuya-kiroku-adapter).
It leases messages with a visibility timeout, handles retries, supports an optional
dead-letter queue (DLQ), and emits OpenTelemetry traces. It is built on
`pgmq-hs` and exposes `Shibuya.Adapter.Pgmq` (plus `.Config` and `.Convert`).
