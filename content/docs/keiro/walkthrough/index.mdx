These are **ordered source tours**: numbered chapters that read keiro's real Haskell source to
teach how each subsystem is built. Read each tour in order. There is one tour per subsystem, plus a
**foundation** tour for the keiki↔keiro core that the others build on and a cross-cutting
**operations** tour for telemetry and migrations.
