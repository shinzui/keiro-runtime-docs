---
title: pgmq
description: A PostgreSQL-native message queue — the queue substrate of the keiro runtime, via pgmq-hs.
---

pgmq is a PostgreSQL-native message queue (the `pgmq` Postgres extension), and
**pgmq-hs** is the first-party Haskell client for it. It is the runtime's _queue
substrate_: create queues, send messages, lease them with a visibility timeout,
then archive or delete — the queue equivalent of what kiroku is for event storage.
Workers consume a pgmq queue through shibuya via the
[shibuya ⇄ pgmq adapter](/docs/integrations/shibuya-pgmq-adapter).
