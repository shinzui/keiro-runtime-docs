## Prerequisites

- A recent **GHC** and **Cabal** (the Haskell toolchain).
- For kiroku: a running **PostgreSQL** instance. kiroku does not run your
  migrations — create its schema first, then point kiroku at the database.

## Getting the libraries

The libraries are distributed as Haskell packages. Add the ones you need to your
project's build configuration:

```text
-- in your .cabal build-depends
build-depends: kiroku, keiro, keiki, shibuya
```

Then see each library's **Tutorials** section to get started.
