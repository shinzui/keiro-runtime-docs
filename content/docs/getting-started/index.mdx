The **keiro runtime** is a family of four Haskell libraries for building
event-sourced systems. This documentation is organized so you can find what you
need quickly, whatever kind of help you are after.

## How these docs are organized

Every page serves exactly one need, following the **Diátaxis** framework:

<Cards>
  <Card title="Tutorials" description="Learning-oriented lessons — start here if you are new." />
  <Card title="How-To Guides" description="Task-oriented recipes for a specific goal." />
  <Card title="Reference" description="Dry, complete facts: APIs, types, configuration." />
  <Card title="Explanation" description="Understanding why things work the way they do." />
</Cards>

Each library repeats these sections, plus a **Cookbook** of short recipes and a
**Code Walkthrough** that tours the real source.
