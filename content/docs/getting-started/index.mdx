The **keiro runtime** is five composable Haskell libraries plus the pg-migrate
schema toolkit. There is no single `keiro-runtime` package: choose the highest
layer that owns your problem, then add only the integrations it needs.

## A four-step path

1. [Choose a library](/docs/getting-started/choosing-a-library) by problem and
   desired abstraction level.
2. Check [compatibility and upgrades](/docs/getting-started/compatibility-and-upgrades)
   before pinning package and adapter versions.
3. Follow [installation](/docs/getting-started/installation), including the
   application-owned database migration plan.
4. Continue with the selected product's tutorial, then use its how-to and
   operations pages for production work.
