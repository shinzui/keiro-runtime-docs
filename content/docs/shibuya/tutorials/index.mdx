title: Tutorials
description: Guided lessons for building and testing a shibuya worker from the core API.
---

These tutorials start from the smallest source-faithful worker shape and then add handler decisions
and deterministic tests. They use the same `Adapter`, handler-facing `Message`, `Handler`, and
`runApp` types that the real adapters use.
