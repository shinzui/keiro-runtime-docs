---
title: "Expose processor metrics"
description: "Publish JSON and Prometheus views of every processor."
---

```haskell
let config =
      defaultConfig
        { enableJSON = True
        , enablePrometheus = True
        , port = 9090
        }

withMetricsServer config (getAppMaster appHandle) useServer
```

JSON:

```bash
curl -s http://127.0.0.1:9090/metrics | jq
curl -s http://127.0.0.1:9090/metrics/orders | jq
```

Prometheus:

```bash
curl -s http://127.0.0.1:9090/metrics/prometheus
```

The JSON view is better for debugging one worker. The Prometheus view is better for alerts and
dashboards.
