---
title: "Run local Jaeger"
description: "Send shibuya OpenTelemetry spans to a local Jaeger collector over OTLP HTTP."
---

Start Jaeger with OTLP enabled:

```bash
jaeger --collector.otlp.enabled=true
```

Use OTLP HTTP port `4318`:

```bash
export OTEL_SERVICE_NAME=shibuya-local
export OTEL_EXPORTER_OTLP_ENDPOINT=http://127.0.0.1:4318
export OTEL_TRACES_SAMPLER=always_on
```

Run the worker with `runTracing tracer`, then open:

```text
http://127.0.0.1:16686
```

Find service `shibuya-local`. Current shibuya spans are named `"<processor> process"`, such as
`"orders process"`, and carry `shibuya.ack.decision`.
