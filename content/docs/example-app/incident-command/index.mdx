The **`incident-command`** service (Haskell package `incident-command`, modules
`IncidentCommand.*`) is the command-side of the emergency: it declares incidents, assigns
commanders, dispatches field resources, triages casualties, orders evacuations, and escalates
critical incidents on a durable timer. When a triage marks a critical patient it emits a *transfer
need* that travels to [Hospital Capacity](/docs/example-app/hospital-capacity) over Kafka.

This subsection is an ordered source tour, one chapter per concern, that reads the real Haskell at
the pinned commit and links each feature back to its canonical keiro reference page. Start with
chapter 00.
