The **`hospital-capacity`** service (Haskell package `hospital-capacity`, modules
`HospitalCapacity.*`) is the receiving side of the emergency: it tracks hospitals, bed capacity,
and supply levels, runs a surge protocol on a durable timer, and holds, confirms, and releases
patient-transfer reservations. It drains reservation work through a **`keiro-pgmq`** background-job
queue — the same example the keiro pgmq pages already use.

This subsection is an ordered source tour, one chapter per concern, that reads the real Haskell at
the pinned commit and links each feature back to its canonical keiro reference page.
