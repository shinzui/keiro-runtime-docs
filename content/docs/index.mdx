The **keiro runtime** is a family of five Haskell libraries for building
event-sourced systems. It has two foundations: **kiroku** is the foundation for
_persistence_ (durable event storage), and **keiki** is the foundation for _pure
semantics_ (law-abiding, IO-free composition). **pgmq** rounds out the family as the
_queue substrate_ — a PostgreSQL message queue that shibuya's workers lease from.

## How they fit together

```mermaid
flowchart TD
    keiro[keiro — framework + workflow engine]
    kiroku[(kiroku — event store)]
    keiki[keiki — pure core]
    shibuya[shibuya — queue processing]
    pgmq[(pgmq — queue substrate)]
    keiro --> kiroku
    keiro --> keiki
    keiro --> shibuya
    shibuya -. via shibuya-pgmq-adapter .-> pgmq
```
